프런트엔드 상담소

프런트엔드 상담소에서는 프런트엔드에 관한 여러 가지 궁금증을 해결해드립니다.

  • TOAST UI,
    이렇게 사용하고 있는데
    맞나요?
  • 프런트엔드 성능을
    높이고 싶은데,
    방법이 없을까요?
  • 프런트엔드를
    제대로 배우고 싶은데
    어떻게 하면
    좋을까요?

JavaScript 잘알못도, 초고수도 괜찮습니다.
준비물은 단 하나, 프런트엔드 개발을 하면서 느꼈던
고민 또는 질문만 있으면 됩니다.

프런트엔드 상담소가 여러분의 고민을 듣고
뼛속까지 시원하게 답해드립니다.

상담은 신청을 받고 추첨으로 진행됩니다.
상담은 아니더라도 TOAST UI 관련 문의나
FE 개발에 대해서라면 언제든지 환영합니다.

시간/장소

14:00~18:00
별도 링크로 비공개 화상 상담 진행
대상

프런트엔드 상담소에 관심 있는 프런트엔드 개발자
상담 주제 예시
  • TOAST UI 제품 사용법 및 트러블 슈팅
  • 프런트엔드 개발 문화(코드 리뷰, 테스트 등 )
  • JavaScript, 프런트엔드 기술 전반
  • 오픈소스 프로젝트
  • 성능
  • NHN FE개발랩 채용 상담
